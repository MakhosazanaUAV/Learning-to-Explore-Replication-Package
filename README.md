# Replication Package: Learning-Based Autonomous MAV Exploration

This repository provides the full replication package supporting the systematic literature review entitled  
**"Learning to Explore: A Systematic Literature Review on Autonomous MAV Exploration in Confined Environments Using Learning-Based Methods."**

The materials in this repository are provided to ensure transparency, reproducibility, and methodological rigor in accordance with PRISMA guidelines.

---

## Contents

### 1. Search Strategy
- Complete search strings used for IEEE Xplore, ScienceDirect, Web of Science, and Scopus
- Database-specific adaptations and Boolean logic
- Search dates and filtering details

### 2. Screening Procedure
- Inclusion and exclusion criteria
- Title and abstract screening rules
- Full-text eligibility assessment guidelines

### 3. AI-Assisted Screening
- Exact ChatGPT prompts used for abstract summarisation
- Example outputs illustrating screening assistance
- Description of human verification steps
- Link to full ChatGPT interaction transcript (Reference [56])

### 4. Study Selection
- Final list of included studies
- Reasons for exclusion at each screening stage
- PRISMA flow information

### 5. Data Extraction
- Extracted study attributes (platform, environment, learning method)
- Reward design and action-space classification tables

### 6. Quality Assurance
- Publication venue information
- Citation counts for all included studies
- Justification of study quality and credibility

---

## Reproducibility Notes

- AI tools were used only as assistive summarisation devices.
- All inclusion/exclusion decisions were verified by human reviewers.
- Database searches can be fully replicated using the provided strings.
