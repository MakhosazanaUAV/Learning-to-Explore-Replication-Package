# Replication Package: Learning-Based Autonomous MAV Exploration

This repository provides the full replication package supporting the systematic literature review entitled  
"Learning to Explore: A Systematic Literature Review on Autonomous MAV Exploration in Confined Environments Using Learning-Based Methods."

The materials in this repository are provided to ensure transparency, reproducibility, and methodological rigor in accordance with PRISMA guidelines.
